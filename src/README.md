# Source Code Directory

This directory contains the core source code for the freight rail arrival delay prediction system.

## Contents
- Data preprocessing scripts
- Feature engineering modules
- Machine learning model training and evaluation code

## Design Principles
- Modular and reusable components
- Clear separation of data processing and modeling logic
- Reproducibility and maintainability

## Usage
Scripts in this directory are designed to be executed independently or integrated into pipelines for training and evaluation.
