# Notebooks Directory

This directory contains Jupyter notebooks used for exploratory data analysis, preprocessing, feature engineering, and model development.

## Contents
- Data understanding and exploratory analysis notebooks
- Data cleaning and preprocessing notebooks
- Model training and evaluation notebooks

## Usage
Notebooks are intended for experimentation, analysis, and reproducibility of research findings.

## Notes
- Notebooks follow a sequential workflow and should be executed in order when possible.
- Finalized logic is migrated to the `src/` directory for maintainability.
