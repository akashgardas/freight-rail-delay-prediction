# Data Directory

This directory contains the datasets used for short-term arrival delay time prediction in freight rail operations.

## Contents
- Raw operational data (arrival times, delays, schedules, etc.)
- Preprocessed or cleaned datasets used for modeling

## Notes
- Due to data confidentiality or licensing constraints, raw datasets may not be publicly shared.
- If datasets are not included, this directory may contain sample data or data schema descriptions.

## Usage
Ensure that data files are placed in this directory before executing preprocessing or modeling scripts.

## Ethical Considerations
All data usage must comply with applicable data privacy, security, and ethical guidelines.
